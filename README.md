# Axel Xiberras — Portfolio

Personal portfolio site for Axel Xiberras, UX/UI Designer based in Paris.

Built as a single-page application (`index.html`) with hash-based routing — no build tools or dependencies required.

## Pages

| Route | Content |
|---|---|
| `/` | Home / Hero |
| `/#projects` | Selected Work (Game Design + Web Design) |
| `/#about` | Bio, Skills, Experience, Education |
| `/#contact` | Contact form |

## Deploy on GitHub Pages

1. Push this repository to GitHub (public)
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose branch: `main` — folder: `/ (root)`
5. Click **Save**
6. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

> GitHub Pages may take 1–2 minutes to build on first deploy. Subsequent pushes are usually faster.

## Local preview

No server needed — just open `index.html` directly in any browser.
